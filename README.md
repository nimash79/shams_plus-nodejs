# Shams Plus - nodejs

Shams Plus bank This is back-end of this project. You can see front-end from [here](https://github.com/nimash79/shams_plus-reactjs)

## Installation

Use [npm](https://npmjs.com/) to install pakcages.

```bash
npm install
```

## Usage

```bash
npm start
```
